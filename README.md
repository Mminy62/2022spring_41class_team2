# 프로그램 세팅 방법

## 버전
- Node : 17.5
- Python : 3.7

## Install Dependencies

### 프로그램 실행 관련
```shell
npm install concurrently
```
### 프론트 엔드 환경
```shell
cd client
npm install .
```
### 벡엔드 환경 설정
```shell
conda create env -n {env_name} python=3.7
cd server
pip install -r requirement.txt
```
<br>

# 프로그램 실행 방법
```shell
conda activate {env_name}
npm run dev
```
